NameDescription
v29encoder This star inputs data and ouputs the coordinates of the CCITT v.29 encoder constlations.
v2b This star converts a byte input vector to a digital stream.
vcm Voltage-controlled multivibrator
vecbit This star converts a byte input vector to a digital stream.
wave This star simulates a wave generator.
wraiff Writes samples from an arbitrary number of input buffers to an AIFF file, which is named as a parameter.
xdco This star produces samples of the outputs every 1/fs seconds. The dco behaves just like an FM modulator. The phase is updated as theta=theta+lambda (integrates the input) *
xnor Function performs logical "xnor" of all its input samples
xor Function performs logical "xor" of all its input samples
xygen Generate xy sweep
zc Zero crossing detector. Outputs impulses at zero crossing.
zdummy This star multiplies the incoming data stream by the parameter coefficient, and outputs the resulting data values.
zero This star sends out a number of zero samples. The only parameter tells how many zero samples to send out.
zlpf This star implements an IIR filter as a recursive equation: y(n)=pole*y(n-1)+(1-pole)*x(n). This implementation produces unity gain at DC.
