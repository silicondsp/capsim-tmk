NameDescription
scattertxt Scatter Probe
scrambler The scrambler and descrambler implemented here,which are self-synchronizing, are given in the CCITT recommendation V35.
sdet Output sine() of input buffer
sdr Compute signal to distortion ratio
secord Second order filter
seqgen This function generates a sequence of bits, which can be used to exercise a data transmission system. Any degree polynomial can be implemented as specified in a parameter array.
server This star models an M/M/1 queue
setsnr Set the Signal to Noise Raio (SNR) by calculating power over window of complex input samples and adding noise.
sine This star generates a sinusoid ( cosine for zero phase). If a second buffer is connected, the quadrature signal is output.
sink This star absorbs data from an arbitrary number of input buffers.
skipold Star skips the first N data points.
skip This star devours the first N values, useful for skipping transient periods
slice This star simulates a decision element for a data receiver.
slidefft This star computes the ONE-SIDED FFT of a complex stream which is overlapped by an amount of (fftl-M) samples for each FFT computation. It outputs ((int)((totalNumberofPoints - fftl)/M)+1)*fftl/2 points. The totalNumberofPoints denotes the number of points in the complex input
sn74ls93 counter with QA connected to CKB
spectrumtxt Spectrum probe (plots both frequency and time domain). Float/Int/Complex input buffers.
spiceprb Spice probe. Used in spice circuit simulation.
spread This star inputs data and spreads it with a sequence from a file.
sqr Square the input buffer
sqrtnyq This star performs square root Nyquist pulse shaping for a baseband transmitter.
