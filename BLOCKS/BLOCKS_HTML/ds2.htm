Main Code



 

while(IT_IN(0))
{

	x = xold + g1*in(0) - g1*p;

	y = yold + g2*xold - g2*p;

	p = delta;
	if(y < 0.0) p = -delta;

	
	if(numberOutputBuffers==1) {
		if(IT_OUT(0)) {
			KrnOverflow("ds2",0);
			return(99);
		}
		OUTF(0,0)=p;
	} else {
		if(IT_OUT(0)) {
			KrnOverflow("ds2",0);
			return(99);
		}
		OUTF(0,0)=p;
		if(IT_OUT(1)) {
			KrnOverflow("ds2",1);
			return(99);
		}
		OUTF(0,0)=y;

	}


	xold = x;
	yold = y;
	
}

This star is a self-contained second order delta sigma modulator.  Ouput 0
is the output of the circuit. Output 1 is the input to the comparator
if connected.
-Parameter one: the gain of the first integrator
-Parameter two: the gain of the second integrator
-Parameter three: the value for delta
